Fuentes en HTML

La marca <font> contiene los siguientes atributos:

ATRIBUTO VALOR
TAMAÑO SIZE = N (N varía de 1 a 7)
COLOR COLOR = RGB (R = ROJO, G = VERDE, B = AZUL)
Cada uno de estos colores toma valores de 00 hasta FF - Sistema Hexadecimal
TIPO FACE = Nombre de la fuente

Ejemplo

<HTML>
<HEAD> <TITLE>EJEMPLO4</TITLE> </HEAD>
<BODY>

<P><FONT SIZE = 5>texto de tamaño 5</FONT></P>

<P><FONT COLOR = FF0000>texto de color rojo</FONT></P>

<P><FONT FACE= "Comic Sans MS">texto con fuente Comic Sans MS</FONT></P>

<P><FONT SIZE = 7 COLOR = 0000FF FACE= "Comic Sans MS">texto con fuente de tamaño 7,
color azul y tipo Comic Sans MS</FONT></P>

</BODY>
</HTML>

¡Inténtalo! Prueba este código en tu navegador y observa cómo se ven los diferentes atributos de las fuentes.
