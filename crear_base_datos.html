Pasos para Crear una Base de Datos en USBwebserver

Sigue estos pasos para crear una base de datos y una tabla en la plataforma MySQL usando USBwebserver:

Reglas

  1. Crear una base de datos con el nombre de base1
  2. Crear una tabla con el nombre de alumnos
  3. En la tabla crear 4 campos: código, nombre, mail, codigocurso
  4. Cada campo tendrá las siguientes características:
    • código - tipo int
    • nombre - tipo varchar con una longitud de 50
    • mail - tipo varchar con una longitud de 70
    • codigocurso - tipo int
