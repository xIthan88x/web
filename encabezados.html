Encabezados en HTML

La marca HN se utiliza para crear encabezados, es decir, títulos y subtítulos. N puede tomar valores de 1 a 6 (<H1> hasta <H6>), siendo <H1> el mayor nivel y <H6> el menor. No es recomendable utilizar esta marca para cambiar el tamaño de la letra.

Ejemplo

<HTML>
<HEAD> <TITLE>EJEMPLO1</TITLE> </HEAD>
<BODY>
<H1>MARCA PARA TITULOS H1</H1>
<H2>MARCA PARA TITULOS H2</H2>
<H3>MARCA PARA TITULOS H3</H3>
<H4>MARCA PARA TITULOS H4</H4>
<H5>MARCA PARA TITULOS H5</H5>
<H6>MARCA PARA TITULOS H6</H6>
</BODY>
</HTML>

¡Inténtalo! Prueba este código en tu navegador y observa cómo se ven los diferentes niveles de encabezados.
