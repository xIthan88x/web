Crear un Formulario en PHP

Crear un formulario en PHP permite recoger datos del usuario y procesarlos en el servidor. Aquí te mostramos cómo hacerlo paso a paso.

Ejemplo de Código

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Formulario de Ejemplo</title>
</head>
<body>
    <h1>Formulario de Ejemplo</h1>
    <form action="procesar_formulario.php" method="post">
        <p>
            <label for="nombre">Nombre:</label>
            <input type="text" name="nombre" id="nombre">
        </p>
        <p>
            <label for="email">Correo Electrónico:</label>
            <input type="email" name="email" id="email">
        </p>
        <p>
            <input type="submit" value="Enviar">
        </p>
    </form>
</body>
</html>

¡Inténtalo! Prueba este código en tu navegador y observa cómo se ve el formulario.

Código para procesar_formulario.php

<?php
$nombre = $_POST['nombre'];
$email = $_POST['email'];

echo "Nombre: " . htmlspecialchars($nombre) . "<br>";
echo "Correo Electronico: " . htmlspecialchars($email);
?>

Este archivo PHP procesa los datos enviados desde el formulario y muestra los valores recibidos.
