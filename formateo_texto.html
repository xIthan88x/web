Formateo de Texto en HTML

En la siguiente tabla se describen las principales marcas para formatear texto:

FORMATO MARCA
NEGRILLA <B> .. </B>
ITALICA <I> .. </I>
SUBRAYADO <U> .. </U>
ENFATIZADO <STRONG> .. </STRONG>

Dentro de la apertura de la marca <MARCA> y su cierre </MARCA> debe escribirse el texto al que se le quiere dar formato.

Ejemplo

<HTML>
<HEAD> <TITLE>EJEMPLO3</TITLE> </HEAD>
<BODY>

<H3>MARCA PARA NEGRILLA</H3>
<P><B>texto en negrilla</B></P>

<H3>MARCA PARA ITALICA</H3>
<P><I>texto en itálica</I></P>

<H3>MARCA PARA SUBRAYADO</H3>
<P><U>texto subrayado</U></P>

<H3>MARCA PARA ENFATIZADO</H3>
<P><STRONG>texto enfatizado</STRONG></P>

</BODY>
</HTML>

¡Inténtalo! Prueba este código en tu navegador y observa cómo se ven los diferentes formatos de texto.
