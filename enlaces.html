Tipos de Enlaces

Existen tres tipos de enlaces principalmente:

  1. A otro sitio web.
  2. A otra página web dentro del mismo sitio.
  3. A otro punto dentro de la misma página.

La marca <A> .. </A> nos permite realizar cualquiera de los tipos de enlace mencionados.

Ejemplo

<HTML>
<HEAD> <TITLE>EJEMPLO11</TITLE> </HEAD>
<BODY>
<P>Enlace a otro sitio web
<BR>
<A HREF = "https://www.dreamworks.com/how-to-train-your-dragon">Como entrenar a tu dragón</A>
<P>Enlace a otra página
<BR>
<A HREF = "pagina5.html">Manejo del color</A>
