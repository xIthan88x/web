Introducción

Para realizar estos ejemplos, necesitarás una imagen llamada imagenprueba.jpg. Asegúrate de tener esta imagen en el mismo directorio que tu archivo HTML.

1. Alineamiento de la Gráfica en la Página

El atributo ALIGN permite alinear una gráfica al lado derecho o izquierdo de la página, para centrar una gráfica se inserta la marca de la imagen dentro de la marca <CENTER> .. </CENTER>.

Ejemplo

<HTML>
<HEAD> <TITLE>EJEMPLO8</TITLE> </HEAD>
<BODY>
<IMG SRC = "imagenprueba.jpg" ALIGN = LEFT>
<BR CLEAR=ALL>
<CENTER>
<IMG SRC = "imagenprueba.jpg">
</CENTER>
<BR CLEAR=ALL>
<IMG SRC = "imagenprueba.jpg" ALIGN = RIGHT>
</BODY>
</HTML>

¡Inténtalo! Prueba este código en tu navegador y observa cómo se alinean las gráficas en la página.

2. Alineamiento de Texto en Relación a la Gráfica

El atributo ALIGN permite alinear el texto arriba, en el medio y abajo de una gráfica.

Ejemplo

<HTML>
<HEAD> <TITLE>EJEMPLO9</TITLE> </HEAD>
<BODY>
<IMG SRC = "imagenprueba.jpg" ALIGN = TOP> Texto en la parte de arriba
<BR CLEAR=ALL>
<BR>
<IMG SRC = "imagenprueba.jpg" ALIGN = MIDDLE> Texto en el medio
<BR CLEAR=ALL>
<BR>
<IMG SRC = "imagenprueba.jpg" ALIGN = BOTTOM> Texto en la parte de abajo
</BODY>
</HTML>

¡Inténtalo! Prueba este código en tu navegador y observa cómo se alinea el texto en relación a las gráficas.

3. Espacios Alrededor de la Gráfica y Tamaño

Los atributos HSPACE y VSPACE definen espacios horizontales y verticales alrededor de una gráfica y los atributos WIDTH y HEIGHT permiten definir el ancho y alto de una gráfica.

Ejemplo

<HTML>
<HEAD> <TITLE>EJEMPLO10</TITLE> </HEAD>
<BODY>
<P ALIGN = JUSTIFY><IMG SRC = "imagenprueba.jpg" ALIGN = LEFT HSPACE = 10 VSPACE = 10> Texto que fluye alrededor de una gráfica respetando un espacio horizontal y vertical. Este texto puede ser más de una linea.
<BR CLEAR=ALL>
<BR>
<IMG SRC = "imagenprueba.jpg" ALIGN = LEFT WIDTH = 100 HEIGHT = 70> Gráfica de tamaño modificado (100 x 70)
</BODY>
</HTML>

¡Inténtalo! Prueba este código en tu navegador y observa los espacios alrededor de las gráficas y su tamaño.

4. Gráfica de Fondo para una Página

El atributo BACKGROUND dentro de la marca <BODY> permite definir una gráfica como fondo de una página, de ser necesario se puede cambiar el color de la letra con el atributo TEXT.

Ejemplo

<HTML>
<HEAD> <TITLE>EJEMPLO12</TITLE> </HEAD>
<BODY BACKGROUND = "imagenprueba.jpg" TEXT = 000000>
<H1>TEXTO EN COLOR NEGRO</H1>
</BODY>
</HTML>

¡Inténtalo! Prueba este código en tu navegador y observa cómo se establece una gráfica como fondo de la página.
