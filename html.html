Lo Principal: ¡Abre tu Blog de Notas!

Para empezar a crear una página web, lo primero que necesitamos es un editor de texto, como el "Blog de Notas" de Windows, donde escribirás tu código HTML.

En HTML, cada documento se puede dividir en dos partes principales:

  1. El Cuerpo: Esta es la parte visible de tu documento, es decir, lo que será mostrado por el navegador.
  2. El Encabezado: Aquí se incluye información sobre el documento, pero no se muestra en la página. Por ejemplo, el título o metadatos.

Un documento HTML básico tiene esta estructura:

<!DOCTYPE html>
<html lang="es">
    <head>
        
    </head>
    <body>
        
    </body>
</html>

El elemento <html> es el más exterior y declara que el texto encerrado es un documento HTML. Dentro de este, encontrarás el <head> y el <body>.

Recuerda, el <body> contiene el contenido visible de la página (como textos, imágenes y enlaces), mientras que el <head> contiene la información relacionada con el documento, como el título de la página y otros datos.
