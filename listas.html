Listas Enumeradas

La marca <OL> .. </OL> se utiliza para crear listas con numeración, los elementos de lista se adicionan con la marca <LI>.

Ejemplo

<HTML>
<HEAD> <TITLE>EJEMPLO5</TITLE> </HEAD>
<BODY>
<P>El proceso de matricula consta de los siguientes pasos:
<OL>
<LI>Preinscripción
<LI>Llenar el formulario de inscripción
<LI>Inscripción
<LI>Entrevista
<LI>Matricula
</OL>
</BODY>
</HTML>

¡Inténtalo! Prueba este código en tu navegador y observa cómo se ven las listas enumeradas.

Listas No Enumeradas

La marca <UL> .. </UL> se utiliza para crear listas sin numeración, los elementos de lista se adicionan con la marca <LI>.

Ejemplo

<HTML>
<HEAD> <TITLE>EJEMPLO6</TITLE> </HEAD>
<BODY>
<P>La universidad antonio nariño ofrece los siguientes programas:
<UL>
<LI>Ingeniería de sistemas
<LI>Psicología
<LI>Contaduría pública
<LI>Ingeniería electrónica
<LI>Tecnología en sistematización de datos
</UL>
</BODY>
</HTML>

¡Inténtalo! Prueba este código en tu navegador y observa cómo se ven las listas no enumeradas.

Listas de Glosario

La marca <DL> .. </DL> se utiliza para crear listas de glosario, cada término se escribe después de la marca <DT>, y la definición seguido de la marca <DD>.

Ejemplo

<HTML>
<HEAD> <TITLE>EJEMPLO7</TITLE> </HEAD>
<BODY>
<P>Terminología de internet:
<DL>
<DD>Red <DT>Conjunto de computadoras interconectadas entre sí con el fin de compartir recursos e información.
<DD>Browser <DT>Aplicación que se utiliza para visualizar páginas web, también se conoce con el nombre de navegador, los más utilizados son el Internet Explorer y el Netscape.
<DD>HTML <DT>Lenguaje de marcas de hypertexto.
</DL>
</BODY>
</HTML>

¡Inténtalo! Prueba este código en tu navegador y observa cómo se ven las listas de glosario.
