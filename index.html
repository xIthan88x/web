¿Qué es HTML?

HTML son las siglas de HyperText Markup Language, que en español significa Lenguaje de Marcado de Hipertexto. Es el lenguaje estándar que se utiliza para crear y estructurar páginas web.

¿Para qué sirve HTML?

  • Organizar el contenido: Define si algo es un título, un párrafo, una lista, una imagen, etc.
  • Crear enlaces: Permite que el usuario navegue de una página a otra.
  • Incluir multimedia: Puedes agregar imágenes, videos, audio y otros elementos.
  • Hacer que las páginas sean accesibles: Ayuda a que personas con discapacidades puedan utilizar la web.

Diferencia entre página estática y dinámica

  • Páginas web estáticas:
    • Contenido fijo: No cambia a menos que se edite el código manualmente.
    • Simples: Fáciles de crear y mantener.
    • Rápidas: Cargan más rápido.
    • Ideales para: Sitios pequeños y sin mucho contenido dinámico.
  • Páginas web dinámicas:
    • Contenido variable: Se actualiza automáticamente.
    • Interactivas: Permiten acciones como comprar o comentar.
    • Personalizadas: Se adaptan a cada usuario.
    • Complejas: Requieren más desarrollo.
    • Ideales para: Sitios grandes, tiendas online, blogs, etc.

Diferencia entre una página web y un portal web

Característica Página web Portal web
Alcance Más limitado Más amplio
Función Proporciona información específica Actúa como puerta de entrada a múltiples recursos
Personalización Mínima Alta (a menudo permite personalizar la experiencia del usuario)
Acceso Generalmente público Puede requerir registro o inicio de sesión
