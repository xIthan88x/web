Atributos y Elementos del Formulario

La marca <FORM> permite crear formularios en HTML. Los atributos principales son:

  • METHOD: Especifica cómo se enviarán los datos al servidor (GET o POST).
  • ACTION: URL del programa que procesará los datos del formulario.
  • NAME: Nombre del formulario.

La marca <INPUT> se utiliza para crear elementos dentro de un formulario. Los principales tipos de elementos son:

  • button: Crea un botón de comando.
  • reset: Crea un botón para borrar los datos.
  • submit: Crea un botón para enviar los datos al servidor.
  • checkbox: Crea casillas de verificación.
  • radio: Crea un grupo de opciones donde solo se puede seleccionar una.
  • password: Crea un campo para ingresar contraseñas.
  • text: Crea un campo de texto de una sola línea.
  • textarea: Crea un campo de texto para varias líneas.
  • select: Crea un campo de selección de opciones.
  • hidden: Crea un elemento oculto.

Los principales eventos que se pueden generar son:

  • onClick: Disparado cuando se hace clic sobre el elemento.
  • onFocus: Disparado cuando un elemento recibe el foco.
  • onBlur: Disparado cuando un elemento pierde el foco.
  • onChange: Disparado cuando el valor de un campo cambia.

Ejemplo de Formulario

<HTML>
<HEAD><TITLE>EJEMPLO14</TITLE></HEAD>
<BODY>
<FORM NAME="frmEncuesta" METHOD="POST" ACTION="/scripts/encuesta.asp">
    <P>Código:</P>
    <INPUT TYPE="text" NAME="txtCodigo" MAXLENGTH="8" SIZE="8">
    <P>Nombre:</P>
    <INPUT TYPE="text" NAME="txtNombre" MAXLENGTH="40" SIZE="40">
    <P>Sexo:</P>
    <INPUT TYPE="radio" NAME="optSexo" VALUE="femenino" CHECKED>Femenino
    <INPUT TYPE="radio" NAME="optSexo" VALUE="masculino">Masculino
    <P>Programa:</P>
    <SELECT NAME="cboPrograma">
        <OPTION VALUE="10">Ingeniería de Sistemas</OPTION>
        <OPTION VALUE="20">Psicología</OPTION>
        <OPTION VALUE="15">Contaduría Pública</OPTION>
    </SELECT>
    <P>Áreas de Estudio:</P>
    <INPUT TYPE="checkbox" NAME="Matematica" VALUE="1">Matemática</INPUT>
    <INPUT TYPE="checkbox" NAME="Humanidades" VALUE="2">Humanidades</INPUT>
    <INPUT TYPE="checkbox" NAME="Tecnica_Profesional" VALUE="3">Técnica Profesional</INPUT>
    <P>Sugerencias:</P>
    <TEXTAREA NAME="txtSugerencias" ROWS="5" COLS="50"></TEXTAREA>
    <P>
    <INPUT TYPE="submit" VALUE="Enviar">
    <INPUT TYPE="reset" VALUE="Limpiar">
</FORM>
</BODY>
</HTML>
